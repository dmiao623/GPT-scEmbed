### GPT-scEmbed

Single-cell transcriptome analysis (scRNA-seq) analysis using Chat-GPT. Based
off of the work of [Chen et. al](https://doi.org/10.1101/2023.10.16.562533) ([repo](https://github.com/yiqunchen/GenePT)).
